## Envoyez des notifications avec le câble USB
<img src="./no.png">

Vous aurez besoin d'adb (Android Debug Bridge) dans le PATH.

- Installez adb et ajoutez-le au PATH
- Lancez le serveur
- Lancez le fichier HTML
- Connectez ensuite votre téléphone **Android** en USB
- Activez le débogage USB dans les paramètres développeur
- Accordez les autorisations, et c'est bon
