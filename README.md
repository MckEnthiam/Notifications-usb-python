## Envoyez vous des notification avec le cable USB
vous aurez besoin de adb (Android Debug Bridge) dans le path 

 - Vous installer adb et vous le metter dans le path
 - vous lancer le serveur
 - et vous lancer le fichier html
 - ensuite vous connecter votre télephone **android** par usb
 - activer le deboguage dans les parametres devellopeur
 - accorder les autorisation et on est bon
